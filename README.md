# KQK

Local SEC company dashboard.

- Find recent `10-K`, `10-Q`, and `8-K` filings.
- Open filings on SEC.
- Chart key SEC company facts.

## Run

```bash
npm install
npm run dev
```

Open http://localhost:3000.

## Check

```bash
npm run typecheck
```

Local data lives in `data/` and is disposable.
